# projeto-sprint5
Dashboard para um aplicativo web em um serviço de nuvem.

## Descrição
Dashboard para analisar dados de anúncios de vendas de carros de forma interativa.

## Funcionalidades
Interface simples com dois botões que geram:
- Visualização de histogramas
- Visualização de gráficos de dispersão

## Tecnologias usadas
- Python
- Streamlit
- Pandas
- Plotly Express
